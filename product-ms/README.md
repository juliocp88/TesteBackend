# Product Microservice

## Installing Libraries
$npm i

## Runnning Docker Database
$docker-compose up -d

## Stopping Docker Database
$docker-compose down

## Runnning Service
$node index.js

## Runnning Tests
$npm run test

## Runnning Unit Tests
$npm run test:unit

## Runnning Feature Tests
$npm run test:feature

## Runnning Lint
$npm run lint
